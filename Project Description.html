Project Description

Using Raspberry Pi technology, the group has decided that Assist in Reflection will only have basic features and simple modules that display user customizable information on it and include the modules Bathroom Checklist and Medicine Tracker to assist the elderly and those in need. The final version of Assist in Reflection would have a touchscreen glass over a monitor connected to a Raspberry Pi.

The Raspberry Pi would be programmed with the default Raspberry Pi OS, and auto-launch Assist in Reflection upon startup. Although there could be scope for more modules to be included, this would be too complex for the task at hand.
